## 自动技能调用规则

开发过程中，完成以下任务后**自动调用**对应技能：

| 任务类型 | 自动调用技能 |
|---------|-------------|
| 代码编写/修改后 | `/simplify` - 审查代码复用、质量、效率 |
| 功能开发完成后 | `/code-review` - 全面代码审查 |
| 前端相关修改 | `/frontend-design` - 前端设计检查 |
| 安全相关代码 | `/security-guidance` - 安全审查 |
| Git 提交前 | `/commit-commands` - 提交规范检查 |

### 执行时机
- 每次文件编辑后自动进行 simplify 审查
- 提交代码前必须经过代码审查
