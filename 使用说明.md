# 智习 - 间隔重复学习系统

## 快速启动

### 首次启动

```bash
cd d:/cc-projects/projects/study
npm install
npm run dev
```

然后打开浏览器访问：**http://localhost:3000**

### 日常启动

每次使用只需运行：

```bash
cd d:/cc-projects/projects/study
npm run dev
```

然后打开浏览器访问：**http://localhost:3000**

---

## 常用命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 启动开发服务器（开发时用）|
| `npm run build` | 构建生产版本 |
| `npm run start` | 运行生产版本（需先 build）|

## 项目结构

```
study/
├── src/
│   ├── app/           # Next.js 页面
│   │   ├── page.tsx       # 首页
│   │   ├── study/         # 学习页面
│   │   ├── library/       # 知识库页面
│   │   └── settings/      # 设置页面
│   ├── components/    # React 组件
│   ├── lib/           # 工具函数和类型
│   └── data/          # 初始数据
├── public/            # 静态资源
└── package.json      # 项目配置
```

---

## 功能概览

- **首页**: 查看学习进度和开始学习
- **学习**: 间隔重复复习卡片，支持章节选择
- **知识库**: 查看和管理所有卡片，支持搜索筛选
- **设置**: 主题切换、语音播报、推送提醒、数据管理
