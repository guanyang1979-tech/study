# 高级经济师备考系统 - 规格文档

## 1. 项目概述

**项目名称**：智习 - 间隔重复学习系统
**项目类型**：Web 应用（学习工具）
**核心功能**：基于艾宾浩斯遗忘曲线的间隔重复学习系统，支持知识点卡片复习、AI 语音播报、自动化推送
**目标用户**：备考高级经济师的个人开发者

### 2.6 响应式断点
- 移动端：< 640px
- 平板：640px - 1024px
- 桌面：> 1024px

## 3. 页面结构

### 3.1 路由结构
```
/                       # 首页/学习页面
/study                  # 学习/复习页面
/library                # 知识库管理
/library/[id]           # 知识库详情/编辑
/settings               # 系统设置
```

### 3.2 首页（/）
- 欢迎区域：显示用户名、学习统计
- 今日任务：今日待复习卡片数量
- 学习进度：本周学习情况图表
- 快速开始：开始学习按钮

### 3.3 学习页面（/study）
- 顶部进度条：显示当前复习进度
- 卡片区域：显示知识点卡片（翻转式）
- 底部操作区：掌握/一般/忘记 三个按钮
- 键盘快捷键支持

### 3.4 知识库页面（/library）
- 章节分类列表
- 卡片搜索功能
- 批量导入/导出
- 卡片预览

### 3.5 设置页面（/settings）
- 主题切换（亮色/深色/跟随系统）
- AI 语音设置（TTS 开关、语速）
- 推送设置（Webhook 配置）
- 数据管理（导入/导出/重置）

## 4. 功能规格

### 4.1 间隔重复算法（SM-2 改进版）
```typescript
// 评分：0-5
// 0: 完全忘记
// 1: 错误但看到答案后想起
// 2: 错误但感觉快想起来了
// 3: 正确但很困难
// 4: 正确且稍有犹豫
// 5: 完全正确

// 算法逻辑：
// - 首次复习：interval = 1天
// - 评分 >= 3: interval = interval * easeFactor
// - 评分 < 3: interval = 1, repetition = 0
// - easeFactor 最小为 1.3
```

### 4.2 卡片交互
- 点击卡片翻转（显示答案）
- 空格键：显示答案
- 1/2/3 数字键：快速评分
- 滑动切换：移动端支持

### 4.3 AI 语音播报（TTS）
- 使用 Web Speech API
- 支持中文语音
- 可调节语速
- 自动播报答案

### 4.4 数据持久化
- LocalStorage 存储学习进度
- 支持导入/导出 JSON 数据
- 预留 Supabase 接口（待后续迁移）
